# Digital Campaign Performance Analytics

End-to-end digital advertising analytics project built with **Google BigQuery (SQL)** and **Google Looker Studio**.

## Live Dashboard
[View Interactive Dashboard]:(https://datastudio.google.com/reporting/d5a09cfc-57b0-457f-9e6a-9f5e1d952fc3)

## Tools
- Google BigQuery — data cleaning and SQL analysis
- Google Looker Studio — interactive dashboard
- SQL — querying, joining, aggregating

##  Dataset
- `Ad_campaign.clean.csv` — 1,000 rows of digital ad performance data
- `campaign_budget.csv` — campaign budget planning data

## 🔍 SQL Queries
12 queries covering creative, audience, and investment analysis including JOINs, CASE WHEN, subqueries, and CTEs.

## Dashboard Pages
1. Campaign Overview — KPIs, CTR by content type, ROI by topic, monthly trend
2. Audience Analysis — Age, gender, location, device, geo map
3. Investment vs Performance — Planned vs actual CTR, budget efficiency
4. Executive Summary — Key findings and recommendations

## Key Insights
- Video drives highest CTR (8.3%) but Image delivers best ROI (1.28)
- Mobile is most cost-efficient device — ROI 1.27 at lowest CPC ($0.77)
- Canada outperforms all markets on ROI (1.39) vs USA (1.04)
- 42% of ads need immediate review with avg ROI of 0.35
